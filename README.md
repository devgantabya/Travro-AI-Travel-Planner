# Travro – AI-Powered Travel Companion 🌍✈️

Travro is a modern travel web application built with **Next.js**, **React**, and **Tailwind CSS**, designed to make travel planning seamless, personalized, and fun. Explore destinations, read travel blogs, get AI-powered suggestions, and book your dream trips all in one place.

---

## 🚀 Features

- **Explore Destinations:** Browse, search, and filter destinations by location, price, and rating.
- **AI-Powered Travel Suggestions:** Personalized recommendations based on user preferences.
- **Travel Blog:** Insightful travel blogs with categories, search, and AI-assisted summarization.
- **Authentication:** Signup, login, and social login (Google) support.
- **Destination Details:** Detailed pages with overviews, pricing, ratings, and reviews.
- **Testimonials & CTA:** Engaging user reviews and call-to-action sections.
- **Responsive UI:** Mobile-friendly and modern design.
- **Contact & About Pages:** Learn about Travro and reach out to the team.

---

## 🛠 Tech Stack

- **Frontend:** Next.js (React)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Image Optimization:** Next.js Image component
- **State Management:** React hooks
